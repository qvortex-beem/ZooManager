Назначение и функционал программы:
- просмотр списка зоопарков
- просмотр полного списка всех животных
- добавление, обновление и удаление зоопарков и животных
- привязка животного к зоопарку и удаление этой привязки

Как запусить?
- создать бд AnimalZooDB в SQL Server и таблицы Zoo, Animal, ZooAnimal
- Открыть решение в Visual Studio
- запустить проект
